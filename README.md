# ATM Simulation System
This is a simple console-based ATM simulation project developed using Python.
# Features
- Check account balance
- Deposit money
- Withdraw money
- Menu-driven system
- Input validation for insufficient balance
# Technologies Used
- Python 3
# How to Run
1. Download the project
2. Open terminal or command prompt
3. Navigate to the project folder
4. Run: python atm.py
